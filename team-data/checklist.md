# 팀장 체크리스트 (조직 기본값)

> 이 파일은 조직 공통 fallback이다. 팀별 `teams/{team_id}/checklist.md`가 있으면 그 팀 것을 우선 적용하고, 없으면 이 기본값을 쓴다.

3대 축: ① 자격증 관리 · ② 휴가 현황 관리 · ③ AS-IS⇒TO-BE 역량 향상
(업무/태스크 추적은 체크리스트 외 보조 도구 — `task-tracker`)

## 📜 자격증 관리

### 월간
- [ ] 팀원별 보유 자격증 현황 파악 (담당: certification-manager)
- [ ] 업무 연관성 있는 자격증 취득 계획 수립 (담당: certification-manager)
- [ ] 자격증 시험 일정 및 신청 마감일 확인 (담당: certification-manager)
- [ ] 합격 시 환급 처리 및 축하 메시지 전달 (담당: certification-manager)

### 분기
- [ ] 자격증 유지보수 활동 현황 점검 (Salesforce 등) (담당: certification-manager)
- [ ] 팀 차원의 자격증 취득 목표 설정 (담당: certification-manager)
- [ ] 자격증 관련 교육 세션 참여 독려 (담당: certification-manager)
- [ ] 회사 파트너십 혜택 활용 방안 검토 (담당: certification-manager)

## 🏖️ 휴가 현황 관리

### 주간
- [ ] 팀원별 연차 사용 현황 모니터링 (담당: leave-manager)
- [ ] 향후 2주간 휴가 계획 파악 (담당: leave-manager)
- [ ] 프로젝트 일정과 휴가 일정 충돌 여부 확인 (담당: leave-manager + assignment-manager)
- [ ] 연차 미사용자 휴가 사용 독려 (담당: leave-manager)

### 월간
- [ ] 잔여 기간 대비 연차 누적 현황 점검 (잔여연차/잔여개월×100 = 100~150% 관리) (담당: leave-manager)
- [ ] 마이너스 연차 발생자 관리 (담당: leave-manager)
- [ ] 팀 전체 휴가 밸런스 조정 (담당: leave-manager)

### 특별 관리
- [ ] 1년 미만 신입사원 연차 소멸 방지 (3월 31일 기준) (담당: leave-manager)
- [ ] 프로젝트 종료 후 휴가 사용 계획 수립 (담당: leave-manager + assignment-manager)
- [ ] 휴가비 지원 (연 100만원) 활용 독려 (담당: leave-manager)

## 📈 AS-IS ⇒ TO-BE 역량 향상 관리

### 현재 상태 파악 (AS-IS)
- [ ] 팀원별 현재 기술 스택 및 역량 수준 평가 (담당: growth-coach)
- [ ] 프로젝트 수행 중 발견된 역량 갭 분석 (담당: growth-coach + assignment-manager)
- [ ] 개인별 강점과 개선 필요 영역 식별 (담당: growth-coach)
- [ ] 업무 만족도 및 성장 욕구 파악 (담당: growth-coach)

### 목표 설정 (TO-BE)
- [ ] 팀원별 6개월/1년 역량 개발 목표 설정 (담당: growth-coach)
- [ ] 회사 비즈니스 방향성과 개인 성장 목표 연계 (담당: growth-coach)
- [ ] 측정 가능한 성과 지표 정의 (담당: growth-coach)
- [ ] 역량 개발을 위한 리소스 및 예산 확보 (담당: growth-coach)

### 실행 및 모니터링
- [ ] 월간 1:1 미팅을 통한 진척도 점검 (담당: oneonone-coach + growth-coach)
- [ ] 교육 프로그램 참여 현황 관리 (담당: growth-coach)
- [ ] 멘토링 및 코칭 기회 제공 (담당: growth-coach)
- [ ] 새로운 프로젝트 투입 시 역량 매칭 고려 (담당: growth-coach + assignment-manager)

### 성과 평가 및 피드백 (분기)
- [ ] 분기별 역량 향상도 평가 (담당: growth-coach)
- [ ] 성공 사례 공유 및 동기부여 (담당: growth-coach)
- [ ] 개선이 필요한 영역에 대한 추가 지원 방안 마련 (담당: growth-coach)
- [ ] 차기 목표 수정 및 업데이트 (담당: growth-coach)
